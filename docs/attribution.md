## Interim logo (Wikimedia Commons)

Lane G3. Three freely-licensed SVG marks pulled from Wikimedia Commons as
the GFTB **interim** logo (chosen by the operator 2026-07-04) (replacing the generic SaturnMark
placeholder). All three verified via the Commons API
(`action=query&prop=imageinfo&iiprop=extmetadata`) on 2026-07-04. Files live
in `static/logo/`.
