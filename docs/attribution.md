# Attribution — interim logo candidates (Wikimedia Commons)

Lane G3. Three freely-licensed SVG marks pulled from Wikimedia Commons as
candidates for the GFTB **interim** logo (replacing the generic SaturnMark
placeholder). All three verified via the Commons API
(`action=query&prop=imageinfo&iiprop=extmetadata`) on 2026-07-04. Files live
in `static/logo/`.

## 1. Bus Silhouette (candidate — primary bus option)

- **File:** `static/logo/bus-silhouette.svg`
- **Source:** https://commons.wikimedia.org/wiki/File:Bus_Silhouette.svg
- **Direct asset:** https://upload.wikimedia.org/wikipedia/commons/b/b6/Bus_Silhouette.svg
- **Author:** unknown ("simple wide-spread figures", dingbat font source)
- **License:** CC0 (public domain dedication) — `LicenseShortName: CC0`
- **Attribution required:** No (CC0), but crediting Wikimedia Commons is
  courteous practice.

## Candidates considered and rejected

- ISO 7001 pictogram set (bus embarkation/disembarkation, airport bus) —
  CC BY-SA/attribution icons but too transit-signage-generic, no brand
  personality.
- Deepin Icon Theme "repair-tools" icon set — GPLv3 licensed; rejected,
  GPL is not an appropriate license for a brand mark (copyleft obligations
  don't fit a logo use case).
- `RiverIcon-Waterfall.svg` — public domain but a very generic/abstract
  riverfloating-diagram glyph, weaker as a standalone brand mark than the
  Delapouite waterfall icon.
- No usable crossed-hammer/wrench or bus+tools combination SVG found on
  Commons under a free license fitting the brand; the search turned up
  nothing beyond the rejected Deepin GPL set.
