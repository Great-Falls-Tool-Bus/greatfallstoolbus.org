# Great Falls Tool Bus

A shared tool library on wheels for Lewiston-Auburn, Maine.

**This is a bus. The shop comes later :)**

## What we need right now

- **A logo & graphics.** We have no logo, no wordmark, no bus art. If you
  design, we would love your help.
- **Tools.** 
  1. Can it travel? (would a cart / case / chest help?)
  2. Are the bits marked as part of the set?
  3. Can it be repaired (model number known, manual findable, someone to ask)?
-  **Tool cell captains** (a person per tool cell who knows the kit).
-  **Sewing cell wants:** a good tailoring iron + ironing table, and a
  serger (Singer / Babylock class).
- **Photos** of tools as they get kitted.

The Great Falls Tool Bus is an unincorporated community project.
