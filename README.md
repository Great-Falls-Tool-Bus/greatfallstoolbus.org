# Great Falls Tool Bus

A shared tool library on wheels for Lewiston-Auburn, Maine.

**This is a bus. The shop comes later :)**

## Where we are

We won the bus — the vehicle itself, not just the idea of one. The site is
feature-complete for launch: tool cells, wants/donate, keyholders, and the
on-site reader for the `discuss@latoolb.us` community board are all live,
served from our own cluster (the old Cloudflare Pages hosting is fully
retired). A review pass is happening this week, then we open the doors wider —
public launch is imminent.

## What we need right now

- **Graphics & a proper wordmark.** We have an interim bus-silhouette glyph in
  the header and favicon; if you design, we would still love your help getting
  past "interim."
- **Tools.**
  1. Can it travel? (would a cart / case / chest help?)
  2. Are the bits marked as part of the set?
  3. Can it be repaired (model number known, manual findable, someone to ask)?
-  **Tool cell captains** (a person per tool cell who knows the kit).
-  **Sewing cell wants:** a good tailoring iron + ironing table, and a
  serger (Singer / Babylock class).
- **Photos** of tools as they get kitted.

The Great Falls Tool Bus is an unincorporated community project.
