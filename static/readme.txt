GREAT FALLS TOOL BUS
====================
A shared tool library on wheels for Lewiston-Auburn, Maine.
This is a bus -- the shop comes later :)

WHAT WE NEED RIGHT NOW
----------------------
* A logo & graphics -- designers wanted!
* Tool donations -- especially complex / large / heavy tools
  with many bits and bobs. Before donating, ask:
    1. Can it travel? (cart / case / chest?)
    2. Are the bits marked as part of the set?
    3. Can it be repaired? (model number, manual, who to ask)
* Tool cell captains -- one person per tool cell who knows the kit.
* Sewing cell wants: a good tailoring iron + ironing table,
  and a serger (Singer / Babylock class).
* Photos of tools as they get kitted.

GET ACCESS
----------
Reach out, a keyholder answers. Keyholders vet requests and coordinate
access privately. The keyholders mailing list and contact form are the
public handoff:

    https://greatfallstoolbus.org
